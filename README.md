## 🧠 Minimax Algorithm Explained

The **Minimax Algorithm** is used in decision-making and game theory.

### 🔹 Key Idea:
- The AI assumes:
  - It plays **optimally**
  - The opponent also plays **optimally**

### 🔹 How it works:
- MAX player → tries to **maximize score**
- MIN player → tries to **minimize score**
- The algorithm explores **all possible moves recursively**

### 🔹 Scoring:
- Win → +10  
- Lose → -10  
- Draw → 0  

👉 The AI picks the move with the **best possible outcome**

---

## 🌳 Decision Tree Visualization

This project shows a **decision tree of all possible moves**.

### What you see:

- 📍 **Root Node** → Current board state  
- 🔽 **Branches** → Possible moves  
- 🟦 **Highlighted path** → AI’s chosen move  
- 🔢 **Score** → Evaluation of that state  
- 🔁 **Type (MAX / MIN)** → Who is playing  

👉 This helps you understand **how AI evaluates future moves step by step**
